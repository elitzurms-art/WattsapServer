# הגדרת שכבת AI בשרת

מדריך קצר להפעלת פיצ'ר ה-AI rescue שנוסף לבוט גמ"ח סקי. מריצים פעם אחת בשרת.

## 1. משיכת הקוד

```bash
cd C:\MCP_WhatsApp     # או נתיב הפרויקט בשרת
git pull
```

## 2. התקנת חבילות

```bash
pnpm install
```

מתקין את `@google/genai` (SDK של Gemini) יחד עם שאר התלויות.

## 3. מפתח API של Gemini

### השגת מפתח (חינם)
1. להיכנס ל-https://aistudio.google.com/apikey
2. להתחבר עם חשבון Google
3. ללחוץ "Create API key" → לבחור פרויקט (או ליצור חדש)
4. להעתיק את המפתח (מתחיל ב-`AIza...`)

### הזנת המפתח
אם אין עדיין קובץ `.env` בתיקייה:

```bash
cp .env.example .env
```

לערוך את `.env` ולהחליף את השורה:

```
GEMINI_API_KEY=AIza_המפתח_האמיתי_שלך
```

## 4. הפעלת הבוט מחדש

לפי איך שהבוט רץ:
- **PM2**: `pm2 restart all`
- **nodemon**: יאתחל אוטומטית על שינוי
- **ידני**: לסגור ו-`node bot.js`

## 5. בדיקה

### בדיקת טריגר מורחב
לשלוח לבוט אחת מהוריאציות הבאות — כולן אמורות לפתוח סשן:
- `גמח סקי`
- `גמ"ח סקי`
- `גמח סקי!`
- `שלום גמח סקי`
- `היי גמ"ח סקי בגולן`

### בדיקת AI rescue
אחרי פתיחת סשן, בתפריט הראשי לנסות הודעה חופשית:
- `אני רוצה להחזיר` → אמור להפוך אוטומטית ל-"2" ולעבור למסך החזרה
- `בטלו לי את הכל` → דומה
- `שריון` → אמור לפתוח את מסך השריון

בלוג של הבוט תראה שורות כמו:
```
🧠 AI rewrite: "אני רוצה להחזיר" → "2"
```

### בדיקת טאב אי-הבנות
לאחר פעולת AI ראשונה, בגיליון ה-Google Sheets יופיע טאב חדש בשם **"אי-הבנות"** עם הכותרות:
חותמת זמן | טלפון | שם משתמש | מצב | טקסט גולמי | מקור | כלי AI | קלט AI | פעולה סופית | הערות

הטאב מתעדכן אוטומטית בכל פעם שה-AI מתערב — שימוש עתידי לשיפור הבוט.

## 6. מה קורה אם חסר המפתח?

אם `GEMINI_API_KEY` לא מוגדר ב-`.env`, הבוט ימשיך לעבוד **כרגיל** (מטפלים קשיחים בלבד, ללא AI). אין שגיאות — פשוט חזרה להתנהגות המקורית.

## 7. תקרות Tier חינמי

המפתח חינמי של Gemini מאפשר:
- **10 בקשות/דקה**
- **500 בקשות/יום**
- **gemini-2.5-flash** (המודל שבחרנו)

לבוט הזה יותר ממספיק גם בימים עמוסים.

## 8. ארכיטקטורה (קצר)

```
bot.js (message event)
    ↓
ai/trigger.js      ← זיהוי "גמ"ח סקי" מורחב (ללא AI)
    ↓
handlers/index.js (handleMessage)
    ↓
ai/rescue.js       ← heuristic: האם המטפל הקשיח יצליח?
    ↓ (אם לא)
ai/intent.js       ← Gemini function calling
    ↓
ai/rescue.js       ← תרגום פלט AI לטקסט שהמטפל הקשיח יבין
    ↓
ai/learning.js     ← רישום לטאב "אי-הבנות" בשיטס
    ↓
handleMessage ממשיך עם msg.body המעודכן
```

## 9. פתרון בעיות

| בעיה | פתרון |
|------|-------|
| `Cannot find module '@google/genai'` | להריץ `pnpm install` |
| `GEMINI_API_KEY is not set in .env` | לבדוק ש-`.env` קיים ומכיל את המפתח |
| `AI call timed out after 12000ms` | בעיית רשת — הבוט יטפל כרגיל ללא AI |
| לא רואים טאב "אי-הבנות" | צריך לפחות פעולת AI אחת מוצלחת. בדוק שחשבון ה-service account שמחובר לשיטס יש לו הרשאת כתיבה |
| שגיאות ב-log על שגיאת מודל | להחליף ל-`gemini-2.5-flash-lite` ב-`ai/intent.js` (זול/זמין יותר) |
