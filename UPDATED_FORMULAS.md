# נוסחאות מעודכנות לגיליון ניהול

## 1. עמודת סטטוס (עמודה C)

**מיקום:** C2 (ואז להעתיק למטה)

```
=MAP(D2:D, LAMBDA(item,
  IF(item="", "",
    LET(
      isBorrowed, COUNTIF('תגובות'!F:H, "*" & item & "*") > COUNTIF('תגובות'!I:I, "*" & item & "*"),
      isReserved, COUNTIF('תגובות'!J:J, "*" & item & "*") > COUNTIF('תגובות'!M:M, "*" & item & "*"),

      IF(AND(isBorrowed, isReserved), "מושאל+משוריין",
        IF(isBorrowed, "מושאל",
          IF(isReserved, "משוריין", "במלאי")
        )
      )
    )
  )
))
```

**הסבר:**
- אם יש שאילה פעילה **וגם** שריון פעיל → "מושאל+משוריין"
- אחרת הלוגיקה כמו קודם

## 2. עמודת שם השואל/משריין (עמודה I)

**מיקום:** I2

```
=IF(
  C2="מושאל",
  IFERROR(
    INDEX(
      'תגובות'!$C:$C,
      MAX(
        FILTER(
          ROW('תגובות'!$C:$C),
          (IFERROR(SEARCH(D2, 'תגובות'!$F:$F), 0) +
           IFERROR(SEARCH(D2, 'תגובות'!$G:$G), 0) +
           IFERROR(SEARCH(D2, 'תגובות'!$H:$H), 0)) > 0
        )
      )
    ),
    "לא נמצא מספר"
  ),
  IF(
    C2="משוריין",
    IFERROR(
      TEXTJOIN(", ", TRUE,
        UNIQUE(
          FILTER(
            'תגובות'!$C:$C,
            IFERROR(SEARCH(D2, 'תגובות'!$J:$J), 0) > 0,
            COUNTIFS('תגובות'!$C:$C, 'תגובות'!$C:$C, 'תגובות'!$J:$J, "*"&D2&"*") >
            COUNTIFS('תגובות'!$C:$C, 'תגובות'!$C:$C, 'תגובות'!$M:$M, "*"&D2&"*")
          )
        )
      ),
      "לא נמצא מספר"
    ),
    IF(
      C2="מושאל+משוריין",
      IFERROR(
        TEXTJOIN(", ", TRUE,
          INDEX(
            'תגובות'!$C:$C,
            MAX(
              FILTER(
                ROW('תגובות'!$C:$C),
                (IFERROR(SEARCH(D2, 'תגובות'!$F:$F), 0) +
                 IFERROR(SEARCH(D2, 'תגובות'!$G:$G), 0) +
                 IFERROR(SEARCH(D2, 'תגובות'!$H:$H), 0)) > 0
              )
            )
          ),
          UNIQUE(
            FILTER(
              'תגובות'!$C:$C,
              IFERROR(SEARCH(D2, 'תגובות'!$J:$J), 0) > 0,
              COUNTIFS('תגובות'!$C:$C, 'תגובות'!$C:$C, 'תגובות'!$J:$J, "*"&D2&"*") >
              COUNTIFS('תגובות'!$C:$C, 'תגובות'!$C:$C, 'תגובות'!$M:$M, "*"&D2&"*")
            )
          )
        ),
        "לא נמצא"
      ),
      ""
    )
  )
)
```

**הסבר:**
- אם "מושאל" → השואל האחרון (כמו קודם)
- אם "משוריין" → כל המשריינים (כמו קודם)
- אם "מושאל+משוריין" → השואל האחרון + כל המשריינים (מופרדים בפסיק)

## 4. תאריך תחילת שריון (עמודה K)

**מיקום:** K2

```
=IF(
  OR(C2="משוריין", C2="מושאל+משוריין"),
  IFERROR(
    TEXTJOIN(", ", TRUE,
      UNIQUE(
        FILTER(
          'תגובות'!$K:$K,
          IFERROR(SEARCH(D2, 'תגובות'!$J:$J), 0) > 0,
          COUNTIFS('תגובות'!$D:$D, 'תגובות'!$D:$D, 'תגובות'!$J:$J, "*"&D2&"*") >
          COUNTIFS('תגובות'!$D:$D, 'תגובות'!$D:$D, 'תגובות'!$M:$M, "*"&D2&"*")
        )
      )
    ),
    ""
  ),
  ""
)
```

**שינוי:** הוספתי `OR(C2="משוריין", C2="מושאל+משוריין")` כדי שיראה תאריכים גם כשמושאל+משוריין.

## הוראות עדכון

1. **גיליון ניהול - עמודת סטטוס (C2):**
   - אם יש נוסחת MAP, החלף אותה עם הנוסחה החדשה
   - אם אין, הכנס את הנוסחה ב-C2

2. **עמודות I, J, K, L:**
   - אם הנוסחאות מתחילות בשורה 2, העתק את הנוסחאות החדשות לשורה 2
   - Google Sheets יעתיק אוטומטית למטה (אם יש AutoFill)
   - או: גרור ידנית את הנוסחה למטה

3. **בדיקה:**
   - צור תגובה חדשה: שאל פריט משוריין
   - בדוק שהסטטוס משתנה ל"מושאל+משוריין"
   - בדוק שהשם והטלפון מציגים את שני האנשים
   - בדוק שהתאריכים נשארים
