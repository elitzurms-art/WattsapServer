# Local WhatsApp-Web.js Fork

This folder contains a local copy of the fixed whatsapp-web.js fork from:
https://github.com/timothydillan/whatsapp-web.js/tree/fix/duplicate-events-and-bindings

## Why Local?

This fork fixes critical issues with duplicate events and bindings in WhatsApp Web.js.
By keeping it local:
- No Git required during installation
- No internet connection needed during npm/pnpm install
- Faster and more reliable installation
- Complete offline capability

## Important

**DO NOT DELETE THIS FOLDER!** It's a required dependency for the project.

## Updating

If you need to update this fork:
1. Download the latest version from the GitHub link above
2. Replace the contents of this folder
3. Run the installation script again
